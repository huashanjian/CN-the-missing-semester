Course Introduction & Philosophy

我们为何教授此课

在传统的计算机科学教育中，您很可能会学习许多高级主题，从操作系统到编程语言再到机器学习。但在许多机构中，有一个基本主题很少被涵盖，而是留给学生自己去学习：计算生态系统的素养。

多年来，我们在麻省理工学院协助教授了几门课程，一次又一次地看到许多学生对他们可用的工具了解有限。计算机的初衷是自动化手动任务，然而学生们经常手动执行重复性任务，或者未能充分利用版本控制和文本编辑器等强大工具。在最好的情况下，这会导致效率低下和时间浪费；在最坏的情况下，这会导致数据丢失或无法完成某些任务等问题。

这些主题并非大学课程的一部分：学生们从未被展示如何使用这些工具，或者至少没有被展示如何高效地使用它们，因此在应该简单的任务上浪费了时间和精力。标准的计算机科学课程缺少了关于计算生态系统的关键主题，而这些主题本可以使学生的生活轻松得多。


计算机教育中缺失的一课

为了帮助弥补这一点，我们开设了一门课程，涵盖了我们认为成为一名高效的计算机科学家和程序员所必需的所有关键主题。这门课程非常务实和实用，它提供了对工具和技术的实践性介绍，您可以立即将其应用于您将遇到的各种情况。该课程于2020年1月麻省理工学院的"独立活动期"（一个为期一个月的学期，以较短的学生自办课程为特色）期间进行。虽然讲座本身只对麻省理工学院的学生开放，但我们将向公众提供所有讲座材料以及讲座的视频录像。

如果这听起来适合您，以下是该课程将教授内容的一些具体示例：


命令行 Shell

如何使用别名、脚本和构建系统自动化常见和重复的任务。不再需要从文本文档中复制粘贴命令。不再需要"一个接一个地运行这15个命令"。不再有"您忘记运行这个东西"或"您忘记传递这个参数"。

例如，快速搜索您的历史记录可以节省大量时间。在下面的示例中，我们展示了与导航 `convert` 命令的 shell 历史记录相关的几个技巧。


版本控制 (Version Control)

如何正确使用版本控制，并利用它来使您免于灾难，与他人协作，并快速查找和隔离有问题的更改。不再需要 `rm -rf; git clone`。不再有合并冲突（好吧，至少更少了）。不再有大块注释掉的代码。不再为如何找到破坏代码的原因而烦恼。不再有"哦不，我们删除了可工作的代码吗？！"。我们甚至会教您如何通过拉取请求为他人的项目做出贡献！

在下面的示例中，我们使用 `git bisect` 来查找哪个提交破坏了单元测试，然后我们使用 `git revert` 来修复它。


文本编辑 (Text Editing)

如何从命令行高效地编辑本地和远程文件，并利用高级编辑器功能。不再需要在文件之间来回复制。不再有重复的文件编辑。

Vim 宏是其最佳功能之一，在下面的示例中，我们使用嵌套的 Vim 宏快速将 html 表格转换为 csv 格式。


远程计算机 (Remote Machines)

在使用 SSH 密钥和终端多路复用时，如何在远程计算机上保持清醒。不再需要仅仅为了同时运行两个命令而打开许多终端。不再需要在每次连接时输入密码。不再仅仅因为互联网断开或不得不重新启动笔记本电脑而丢失所有内容。

文件查找 (Finding Files)

如何快速找到您要查找的文件。不再需要在项目中单击文件，直到找到包含所需代码的文件为止。

在下面的示例中，我们使用 `fd` 快速查找文件，并使用 `rg` 快速查找代码片段。我们还使用 `fasd` 快速 `cd` 和 `vim` 最近/常用的文件/文件夹。


数据整理 (Data Wrangling)

如何直接从命令行快速轻松地修改、查看、解析、绘制和计算数据和文件。不再需要从日志文件中复制粘贴。不再需要手动计算数据的统计信息。不再需要电子表格绘图。


虚拟机 (Virtual Machines)

如何使用虚拟机尝试新的操作系统，隔离不相关的项目，并保持主机的清洁整齐。不再会在进行安全实验时不小心损坏您的计算机。不再有数百万个版本不同的随机安装的软件包。


安全 (Security)

如何在互联网上不立即向世界泄露您的所有秘密。不再需要自己想出符合疯狂标准的密码。不再有不安全的开放 WiFi 网络。不再有未加密的消息传递。


总结 (Conclusion)

这些以及更多内容将在12个课程讲座中涵盖，每个讲座都包含一个练习，让您自己更熟悉这些工具。如果您等不及一月份，也可以看看我们去年IAP期间开设的 Hacker Tools 的讲座。它是这门课的前身，涵盖了许多相同的主题。

我们希望在一月份见到您，无论是线上还是线下！
